# MATLAB Engineering Portfolio

A collection of MATLAB engineering tools and computational projects for electrical engineering applications.

## Week 1 Projects

### 1. Ohm's Law Calculator
Calculates voltage, current, or resistance using Ohm’s Law.

### 2. Direct Current (DC) Power Calculator)
Computes power, voltage, or current using electrical power formulas.

### 3. Voltage vs Current Plotter
Visualizes the linear electrical relationship between voltage and current.

### 4. Energy Consumption Estimator
Estimates electrical energy consumption in Wh and kWh.
