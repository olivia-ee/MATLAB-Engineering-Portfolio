# Ohm's Law Calculator

## Description
A MATLAB script that calculates voltage, current, or resistance using Ohm's Law.

## Formula
V = I × R

## Features
- Menu-based input
- Calculates unknown electrical variable
- Formatted engineering output

## MATLAB Concepts Used
- input()
- if / elseif / else
- fprintf()
