# Energy Consumption Estimator

## Description
A MATLAB engineering script that estimates electrical energy consumption based on appliance power rating, operating hours, and number of days used.

## Formula Used
Energy (Wh) = Power × Hours × Days

Conversion:
kWh = Wh / 1000

## Features
- User input-based estimation
- Watt-hour computation
- Kilowatt-hour conversion
- Consumption classification
- Professional output formatting

## MATLAB Concepts Used
- input()
- arithmetic operations
- fprintf()
- if / else
- engineering calculations

## Purpose
This project demonstrates practical energy analysis and introduces real-world electrical usage estimation.

## Output Example
The script displays total energy consumption in both Wh and kWh and classifies usage level.
