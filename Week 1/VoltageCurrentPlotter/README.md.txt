# Voltage vs Current Plotter

## Description
A MATLAB script that generates a Voltage versus Current graph using Ohm’s Law.

## Formula Used
V = I × R

## Features
- User-defined resistance input
- Automatic current vector generation
- Voltage calculation using vector operations
- Graph plotting
- Axis labeling
- Grid display
- Custom graph styling

## MATLAB Concepts Used
- input()
- vector generation
- array operations
- plot()
- xlabel()
- ylabel()
- title()
- grid on

## Purpose
This project demonstrates MATLAB data visualization and electrical system behavior analysis through graphical representation.

## Output Example
The script plots a straight-line graph showing the linear relationship between voltage and current.
