# Direct Current Power Calculator

## Description
A MATLAB script that calculates power, voltage, or current using the formula for direct current (DC) power.

## Formula
P = V x I

## Features
- Menu-based input
- Calculates unknown electrical variable
- Formatted engineering output

## MATLAB Concepts Used
- input()
- if / elseif / else
- fprintf()
